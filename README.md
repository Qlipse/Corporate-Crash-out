# Corporate-Crash-out
2D Pixel Turn-based game made in Unity that will help players get a feeling of corporate through Pokémon-esque gameplay.

## Features
- Turn-based combat
- Pixel-Art
- Tile movement
- Pokémon-esque feel
- More TBA
